判斷方法

HTTP Access Log

看自己的伺服器是否有收到 Request 有可能對方的防火牆有檔 http 對外

DNS Log

大部分的防火牆不會檔 DNS 查詢

返回內容

透過 Banner, Title, Content 等資訊來辨認
