## Alternate query types 備份查詢類型
### cache
`cache:www.csie.fju.edu.tw`
查詢某網頁在google中的cache

### link
`link:www.csie.fju.edu.tw`
查詢所有和網頁有連結的網頁

### related
`related:www.csie.fju.edu.tw`
列出所有和查詢網頁類似的網頁

### info
`info:hoschoc.com`
列出某網站在Google上存有哪些資訊
![](https://i.imgur.com/qlFMcUi.png)

## Other
### define
`define:赫蘿`
尋找某字詞在網路上的定義

## Query modifiers 查詢修飾符
### site
`Usage:"關鍵字 site:網址"`
查詢指定網站內的網頁
`site:tw`
也可以指定國籍

### intext
`Intext:台灣`
只想在網頁內文中尋找資料

### intitle
只想在網頁標題中尋找資料

### inurl
尋找指定的字串在網址列當中

### Filetype
`filetype:pdf`
`filetype:mp3`
